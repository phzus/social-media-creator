## Fluxo de Trabalho

### Antes de Qualquer Conteudo
1. Verificar se COMUNICACAO.md do projeto existe (tom, linguagem, personalidade)
2. Verificar Brandbook do @picasso (cores, fontes, estilo visual)
3. Alinhar pilares de conteudo com @theboss
4. Consultar dados do @analyzer (avatar, concorrencia, objecoes)
5. Se conteudo derivado: verificar peca pilar com @claudinho

### Calendario Editorial Semanal
1. Definir pilares de conteudo com @theboss (3-5 pilares)
2. Mapear temas da semana por pilar
3. Aplicar 4A Framework em cada tema (4 angulos x tema)
4. Distribuir nos formatos ideais por dia
5. Definir Stories diarios (3-7 por dia)
6. Criar hashtag sets por pilar
7. Entregar calendario completo com formato + horario + hashtags

### Atomizacao de Conteudo Pilar
1. Receber conteudo pilar (live, aula, podcast, newsletter)
2. Identificar 5-8 insights/frameworks centrais
3. Aplicar Piramide Reversa: pilar → medio → micro → nano
4. Adaptar NATIVAMENTE para cada plataforma (formato, tom, comprimento)
5. Gerar 30+ pecas derivativas com hook proprio em cada uma
6. Organizar por plataforma e prioridade de postagem
7. Entregar batch completo pronto para producao

### StorySeller (Stories que Vendem)
1. Definir produto/oferta a ser vendido
2. Criar arco narrativo de 5-7 Stories:
   - Story 1: Bastidor intrigante (curiosidade)
   - Story 2: Contexto do problema (identificacao)
   - Story 3-4: Jornada/solucao (valor)
   - Story 5: Resultado/prova (credibilidade)
   - Story 6: Revelacao da oferta (naturalidade)
   - Story 7: CTA sutil (sticker link, resposta)
3. NUNCA parecer anuncio — parecer conversa autentica
4. Mix de formatos: texto, foto, video curto, enquete

### Reciclagem de Conteudo (730-Day Library)
1. Analisar metricas de conteudo existente (engajamento, alcance, saves)
2. Classificar em Tiers:
   - Tier 1 (top 10%): Repost identico
   - Tier 2 (top 30%): Refresh — novo hook, mesmo corpo
   - Tier 3 (top 50%): Rewrite — mesma ideia, formato diferente
   - Tier 4 (bottom 50%): Arquivar — nao reusar
3. Gerar versoes recicladas com novos hooks e formatos
