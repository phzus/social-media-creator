# Checklist: Lancamento de Campanha

> Verificacao antes de abrir carrinho / ativar campanha
> Responsavel: @theboss + @rafa
> Usar ANTES de ativar qualquer campanha paga ou lancamento

---

## Estrategia

```
☐ Briefing Estrategico APROVADO pelo Anthony?
☐ Briefing de Copy COMPLETO (5 partes)?
☐ Tipo de funil DEFINIDO e workflow selecionado?
☐ Oferta ESTRUTURADA e validada (9 elementos Hormozi)?
☐ Metas de campanha definidas (leads, vendas, ROAS)?
☐ Cronograma com datas-chave definido e comunicado?
```

## Copy & Conteudo

```
☐ Todos os emails da sequencia escritos e revisados?
☐ Copy de paginas (captura, vendas, obrigado) finalizada?
☐ Copy de anuncios (imagem + video) finalizada?
☐ Mensagens WhatsApp/Telegram prontas?
☐ Scripts de CPL/Webinar/Live prontos (se aplicavel)?
☐ Copy passou pelo checklist de revisao (copy-review)?
☐ Tom de voz CONSISTENTE com COMUNICACAO.md?
```

## Criativos & Design

```
☐ Criativos de anuncios produzidos (min 3 variacoes)?
☐ Formatos corretos por canal (1080x1080, 1080x1920, etc)?
☐ Slides/materiais de evento prontos (se aplicavel)?
☐ Identidade visual consistente entre pecas?
☐ CTA claro e legivel em todos os criativos?
```

## Paginas & Tecnologia

```
☐ Pagina de captura publicada e testada?
☐ Pagina de vendas publicada e testada?
☐ Pagina de obrigado publicada e testada?
☐ Checkout/pagamento configurado e testado (compra teste)?
☐ Pixel de rastreamento instalado (Meta, Google)?
☐ Eventos de conversao configurados e disparando?
☐ Paginas responsivas (mobile + desktop)?
☐ Velocidade de carregamento < 3 segundos?
```

## Trafego

```
☐ Campanhas criadas no gerenciador (Meta/Google/TikTok)?
☐ Publicos configurados (frio, morno, quente)?
☐ Orcamento diario definido e aprovado?
☐ Estrutura de campanhas segue o plano de midia?
☐ UTMs configurados em todos os links?
☐ Link de rastreamento testado (clicou → chegou)?
```

## Automacoes

```
☐ Fluxo de boas-vindas ativo e testado?
☐ Tags/segmentacao configuradas corretamente?
☐ Sequencia de emails automatica agendada?
☐ Notificacoes para equipe configuradas (novo lead, venda)?
☐ Integracao formulario → CRM/email funcionando?
☐ Automacoes passaram pelo checklist de deploy?
```

## Equipe & Processos

```
☐ Squad comunicado sobre datas de lancamento?
☐ SDR preparado para receber leads (se high ticket)?
☐ Closer preparado com script atualizado (se aplicavel)?
☐ CS preparado para onboarding pos-venda?
☐ Grupo/canal de comunicacao da equipe ativo?
☐ Metricas de monitoramento configuradas (dashboard)?
```

## Validacao Final

```
☐ Teste completo do funil de ponta a ponta (lead → compra)?
☐ E-mail de teste enviado e verificado (formatacao, links)?
☐ Todos os links funcionando (sem 404)?
☐ Backup do conteudo salvo?
☐ Anthony aprovou o lancamento?
☐ Data e hora de ativacao confirmadas?
```

---

**GATE: So ativar campanha quando TODOS os itens aplicaveis estiverem marcados.**

> Itens nao aplicaveis ao tipo de funil devem ser marcados como N/A com justificativa.
