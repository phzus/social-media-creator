# Checklist: Pre-Desafio (Challenge)

> Verificacao antes de iniciar um desafio de 5-7 dias
> Responsavel: @theboss + @claudinho + @salazar
> Usar 48h ANTES do inicio do desafio
> Referencia: Challenge Secrets (Pedro Adao + Russell Brunson)

---

## Estrategia do Desafio

```
☐ Tema/Big Idea do desafio definido e validado?
☐ Duracao definida (5, 6 ou 7 dias)?
☐ Transformacao prometida clara e alcancavel no prazo?
☐ Estrutura diaria definida (missao + live)?
☐ Cada dia tem um "micro win" concreto?
☐ Dia do pitch definido (tipicamente dia 5 ou 6)?
☐ Oferta principal estruturada?
☐ Conexao clara: desafio → oferta (resultado natural)?
```

## Conteudo Diario

```
☐ Dia 1 — Missao + script da live prontos?
☐ Dia 2 — Missao + script da live prontos?
☐ Dia 3 — Missao + script da live prontos?
☐ Dia 4 — Missao + script da live prontos?
☐ Dia 5 — Missao + script da live + pitch prontos?
☐ Dia 6 — Follow-up + urgencia (se 6 dias)?
☐ Dia 7 — Fechamento + ultima chance (se 7 dias)?
☐ Materiais de apoio por dia (PDFs, planilhas, templates)?
☐ Mensagens de grupo por dia escritas e revisadas?
```

## Grupo (WhatsApp/Telegram)

```
☐ Grupo criado e configurado?
☐ Descricao do grupo com regras claras?
☐ Mensagem de boas-vindas automatica?
☐ Admins/moderadores definidos?
☐ Regras do desafio publicadas?
☐ Template de postagem de missao padronizado?
☐ Automacao de entrada configurada (@salazar)?
☐ Automacao de lembrete diario configurada?
```

## Captacao

```
☐ Pagina de inscricao publicada e testada?
☐ Formulario funcionando (nome + email + WhatsApp)?
☐ Redirect para grupo apos inscricao?
☐ Email de boas-vindas com instrucoes configurado?
☐ Campanhas de ads ativas (se trafego pago)?
☐ Emails para base propria enviados?
☐ Posts organicos de convite publicados?
☐ Meta de inscritos definida?
```

## Emails & Automacoes

```
☐ Email de boas-vindas + link do grupo?
☐ Email pre-desafio (expectativa + regras)?
☐ Email diario de missao (D1 a D5/D7)?
☐ Email de live do dia (lembrete 1h antes)?
☐ Email de oferta (dia do pitch)?
☐ Email de urgencia pos-pitch?
☐ Email de ultima chance?
☐ Automacoes de tag por engajamento (@salazar)?
```

## Tecnico

```
☐ Plataforma de live definida (YouTube/Zoom/IG)?
☐ Horario das lives definido e comunicado?
☐ Camera + mic + internet testados?
☐ Gravacao das lives configurada?
☐ Area de membros/conteudo configurada (se aplicavel)?
☐ Pagina de vendas publicada?
☐ Checkout testado?
```

## Engajamento & Gamificacao

```
☐ Sistema de pontos/reconhecimento definido?
☐ Premios para participantes engajados?
☐ Hashtag ou formato de postagem de missao?
☐ Mecanismo de accountability (duplas, grupos menores)?
☐ Templates de celebracao para compartilhar resultados?
```

## Validacao Final

```
☐ Dry-run do Dia 1 completo (live + missao + grupo)?
☐ Fluxo testado: inscricao → grupo → emails → live → missao?
☐ Conteudo de TODOS os dias finalizado?
☐ Squad alinhado (quem faz o que, quando)?
☐ @metrics configurado para monitorar diariamente?
☐ Anthony aprovou conteudo e oferta?
```

---

**GATE: Nao iniciar desafio sem conteudo de TODOS os dias prontos e testados.**

> E NON-NEGOTIABLE ter todo o conteudo pronto antes do Dia 1. Nao improvisar.

---

## Metricas de Sucesso

| Metrica | Meta |
|---------|------|
| Inscritos no desafio | Meta definida |
| Entrada no grupo | > 70% dos inscritos |
| Completaram missao D1 | > 40% |
| Presenca media nas lives | > 30% |
| Engajamento no grupo | Posts diarios dos participantes |
| Conversao (inscritos → compradores) | 3-10% |
| Conversao (engajados → compradores) | 10-25% |
