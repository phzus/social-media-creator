# Checklist: Revisao de Copy

> Verificacao de qualidade de TODA peca de copy antes de publicar
> Responsavel: @theboss (revisao) + @claudinho (autor)
> Usar ANTES de aprovar qualquer peca de copy para producao

---

## Fundamentos

```
☐ Big Idea presente e clara?
☐ Mecanismo Unico mencionado ou implicito?
☐ Promessa principal esta no headline/abertura?
☐ Copy fala com UMA pessoa (voce), nao com muitas?
☐ Tom de voz consistente com COMUNICACAO.md?
☐ Linguagem do avatar (palavras que ELE usa)?
```

## Estrutura

```
☐ Hook/abertura prende atencao nos primeiros 3 segundos?
☐ Transicao logica entre secoes (sem saltos)?
☐ CTA claro e especifico (o que fazer, como, quando)?
☐ CTA repetido em pontos estrategicos?
☐ Estrutura segue o framework adequado ao tipo de peca?
☐ Hierarquia visual facilita leitura rapida (scan)?
```

## Persuasao (Principios de Kennedy)

```
☐ Dor antes do prazer (agitar antes de resolver)?
☐ Prova social presente (depoimentos, numeros, resultados)?
☐ Autoridade do expert demonstrada (nao apenas afirmada)?
☐ Especificidade (numeros concretos, nao vagos)?
☐ Urgencia/escarcidade REAL (nao fabricada)?
☐ Garantia que remove risco?
☐ Razao logica para a oferta ("o motivo de...")?
```

## Avatar & Empatia

```
☐ Dor principal do avatar articulada com precisao?
☐ Desejo profundo conectado (nao superficial)?
☐ Objecoes principais endereçadas na copy?
☐ Leitor se identifica ("isso parece que foi escrito pra mim")?
☐ Estado emocional mapeado corretamente por fase do funil?
```

## Oferta

```
☐ Oferta clara (o que recebe, quanto custa)?
☐ Value stack apresentado antes do preco?
☐ Ancoragem de preco feita corretamente?
☐ Bonus conectados a objecoes especificas?
☐ Garantia apresentada ANTES do preco final?
☐ Escarcidade/urgencia com deadline real?
```

## Tecnico

```
☐ Ortografia e gramatica revisados?
☐ Links funcionando (testar cada um)?
☐ Personalizacao (nome, campos dinamicos) funcionando?
☐ Formatacao correta para o canal (email, WhatsApp, pagina)?
☐ Comprimento adequado para o canal e fase?
☐ Subject line/assunto testavel (email)?
☐ Preview text configurado (email)?
```

## Compliance & Etica

```
☐ Nenhuma promessa que nao pode ser cumprida?
☐ Nenhum depoimento fabricado ou exagerado?
☐ Nenhuma metrica inventada?
☐ Disclaimer presente onde necessario?
☐ Linguagem respeitosa (sem manipulacao abusiva)?
```

## Revisao Especifica por Tipo

### Email
```
☐ Subject line com < 50 caracteres?
☐ Preview text configurado?
☐ Um unico CTA principal?
☐ Funciona sem imagens (alt text)?
```

### WhatsApp
```
☐ Mensagem curta e direta (max 300 palavras)?
☐ Emojis usados com moderacao?
☐ Link encurtado e rastreavel?
☐ Tom conversacional (nao corporativo)?
```

### Pagina de Vendas
```
☐ Above the fold: headline + sub + CTA + prova?
☐ Secoes na ordem correta (16 blocos)?
☐ FAQ com objecoes reais?
☐ Multiplos CTAs ao longo da pagina?
```

### Anuncio
```
☐ Hook nos primeiros 3 segundos (video) ou primeira linha (imagem)?
☐ Copy primaria < 125 caracteres (Meta)?
☐ CTA do anuncio alinhado com CTA da pagina?
☐ Sem texto excessivo na imagem (regra dos 20%)?
```

---

**GATE: Nenhuma peca de copy vai para producao sem TODOS os itens fundamentais marcados.**

> Itens especificos por tipo sao obrigatorios apenas para o tipo correspondente.

---

## Score de Qualidade

| Faixa | Score | Acao |
|-------|-------|------|
| Excelente | 90-100% | Aprovar para producao |
| Bom | 75-89% | Ajustes menores, aprovar apos correcao |
| Medio | 60-74% | Reescrever secoes fracas |
| Fraco | < 60% | Reescrever do zero |
