# Checklist: Pre-Call de Vendas

> Verificacao antes de CADA call de vendas
> Responsavel: @andreza (closer) + @danilo (SDR, passagem)
> Usar ANTES de iniciar qualquer call de vendas

---

## Informacoes do Lead (Pre-Call — 15 min antes)

```
☐ Nome completo e como prefere ser chamado?
☐ Empresa/nicho e cargo/funcao?
☐ PASSAGEM-LEAD.md recebida do @danilo?
☐ Score BANT revisado (Budget, Authority, Need, Timeline)?
☐ Classificacao do lead (HOT/WARM/COLD)?
☐ Historico de interacoes anteriores revisado?
☐ Origem do lead (qual campanha/anuncio/fonte)?
```

## Preparacao do Closer

```
☐ Script de vendas atualizado e na tela?
☐ Oferta e value stack memorizados?
☐ Preco, condicoes de pagamento e garantia memorizados?
☐ 6 principais objecoes e rebuttals prontos?
☐ Cases/depoimentos relevantes para este perfil?
☐ Perguntas de discovery personalizadas para este lead?
```

## Ambiente & Tecnico

```
☐ Ambiente silencioso e sem interrupcoes?
☐ Camera funcionando (se video call)?
☐ Microfone testado?
☐ Internet estavel?
☐ Link da call enviado ao lead com antecedencia?
☐ Plataforma testada (Zoom/Google Meet/etc)?
☐ CRM aberto na ficha do lead?
```

## Materiais de Apoio

```
☐ Link de pagamento pronto para enviar?
☐ Proposta/apresentacao pronta (se aplicavel)?
☐ Contrato/termos prontos (se aplicavel)?
☐ Bonus ou condicao especial autorizada por @theboss?
☐ Formulario de fechamento acessivel?
```

## Durante a Call — Checkpoints

```
☐ Rapport: Quebrar gelo (2-3 min, conexao pessoal)?
☐ Discovery: Perguntas abertas (escuta 70%, fala 30%)?
☐ Identificar dor PRINCIPAL e desejo PROFUNDO?
☐ Confirmar que tem autoridade de decisao?
☐ Confirmar timeline (urgencia real)?
☐ Transicao natural para apresentacao da oferta?
☐ Apresentar value stack ANTES do preco?
☐ Ancoragem de preco (valor total → preco real)?
☐ Garantia ANTES do preco final?
☐ Tentativa de fechamento (nao ter medo de pedir)?
☐ Se objecao: escutar → validar → recontextualizar?
☐ Proximo passo CLARO antes de desligar?
```

## Pos-Call

### Se FECHOU:
```
☐ Link de pagamento enviado imediatamente?
☐ Confirmacao de pagamento recebida?
☐ CRM atualizado (status: FECHADO)?
☐ @carla notificada para onboarding?
☐ @theboss notificado da venda?
☐ Mensagem de boas-vindas enviada ao cliente?
```

### Se NAO FECHOU:
```
☐ CRM atualizado (status + motivo)?
☐ Objecao principal registrada?
☐ Follow-up agendado (data + canal)?
☐ Sequencia de follow-up ativada?
☐ Lead reclassificado se necessario?
```

### Se NO-SHOW:
```
☐ Tentativa de contato imediata (WhatsApp)?
☐ Reagendamento proposto (2 opcoes de horario)?
☐ CRM atualizado (no-show)?
☐ Cadencia de reativacao ativada?
```

---

**GATE: Nao iniciar call sem ter revisado as secoes "Informacoes do Lead" e "Preparacao do Closer".**
