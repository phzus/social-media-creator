# Checklist: Formacao de Squad

> Verificacao antes de liberar o squad para execucao
> Responsavel: @theboss
> Usar ANTES de iniciar qualquer producao

---

## Pre-Requisitos

```
☐ Briefing Estrategico APROVADO pelo Anthony?
☐ Briefing de Copy preenchido (5 partes)?
☐ Tipo de funil DEFINIDO e validado?
☐ Oferta ESTRUTURADA (9 elementos Hormozi)?
☐ Codigo do projeto definido (ex: BIZU)?
```

## Squad

```
☐ Agentes selecionados conforme tipo de funil?
☐ Responsabilidades definidas por agente?
☐ Dependencias mapeadas (quem depende de quem)?
☐ Documento SQUAD.md preenchido?
☐ Nenhum agente com sobrecarga (max 2 projetos simultaneos)?
```

## Estrutura

```
☐ Estrutura de pastas criada em projects/{expert}/?
☐ Pastas seguem o template do tipo de funil?
☐ Taxonomia de nomenclatura definida?
☐ Cronograma com datas-chave definido?
```

## Documentos Base

```
☐ COMUNICACAO.md (tom de voz) preenchido?
☐ AVATAR.md (publico-alvo) preenchido?
☐ OFERTA.md preenchido?
☐ FUNIL.md preenchido?
```

## Validacao Final

```
☐ Todos os inputs necessarios estao disponiveis?
☐ Nenhuma dependencia pendente bloqueia o inicio?
☐ Anthony aprovou o squad e cronograma?
☐ Squad comunicado e alinhado?
```

---

**GATE: So liberar execucao quando TODOS os itens estiverem marcados.**
