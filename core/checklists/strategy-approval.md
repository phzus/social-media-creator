# Checklist: Aprovacao de Estrategia

> Verificacao antes de aprovar qualquer estrategia de campanha
> Responsavel: @theboss (propoe) + Anthony (aprova)
> Usar ANTES de iniciar execucao de qualquer projeto

---

## Viabilidade do Projeto

```
☐ Expert/produto tem potencial de mercado validado?
☐ Publico-alvo definido com clareza (AVATAR.md)?
☐ Tamanho do mercado enderecavel estimado?
☐ Concorrencia mapeada (diretos e indiretos)?
☐ Diferencial competitivo identificado?
```

## Oferta

```
☐ Oferta segue framework Hormozi (9 elementos)?
☐ Equacao de Valor positiva (Dream Outcome × Likelihood / Time × Effort)?
☐ Preco posicionado corretamente na faixa do mercado?
☐ Garantia definida e viavel?
☐ Value stack com pelo menos 3 bonus estrategicos?
☐ Teste de oferta: "seria burrice recusar"?
```

## Funil & Estrategia

```
☐ Tipo de funil adequado ao cenario (ver tabela de decisao)?
☐ Fases do funil mapeadas e sequenciadas?
☐ Metricas-alvo por fase definidas (com benchmarks)?
☐ Canais de trafego selecionados e justificados?
☐ Orcamento de trafego definido e aprovado?
☐ Timeline realista para o tipo de funil?
```

## Recursos

```
☐ Budget total do projeto definido?
☐ ROI projetado justifica o investimento?
☐ Squad necessario disponivel (sem sobrecarga)?
☐ Ferramentas/plataformas necessarias disponiveis?
☐ Expert tem disponibilidade para gravar/participar?
```

## Riscos

```
☐ Principais riscos identificados (min 3)?
☐ Plano de mitigacao para cada risco?
☐ Cenario pessimista calculado (e ainda vale a pena)?
☐ Plano B definido se estrategia principal falhar?
☐ Dependencias externas mapeadas?
```

## Documentacao

```
☐ BRIEFING-ESTRATEGICO completo e revisado?
☐ BRIEFING-COPY completo (5 partes)?
☐ COMUNICACAO.md preenchido?
☐ OFERTA.md preenchido?
☐ FUNIL.md preenchido?
```

## Decisao

```
☐ Apresentacao da estrategia feita ao Anthony?
☐ Perguntas e duvidas esclarecidas?
☐ Anthony APROVOU a estrategia? (GO / NO-GO)
☐ Se NO-GO: pontos de ajuste listados?
☐ Se GO: data de inicio definida?
```

---

**GATE: Nenhuma execucao inicia sem aprovacao explicita do Anthony.**

---

## Criterios de Aprovacao

| Criterio | Peso | Minimo |
|----------|------|--------|
| Viabilidade de mercado | 25% | GO |
| Qualidade da oferta | 25% | GO |
| Adequacao do funil | 20% | GO |
| Recursos disponiveis | 15% | GO |
| Risco aceitavel | 15% | GO |

**Resultado: TODOS os criterios devem ser GO para aprovar.**
