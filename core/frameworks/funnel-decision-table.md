## Tabela de Decisão — Qual Funil Usar?

| Cenário | Funil Recomendado | Alternativa | Por que |
|---------|-------------------|-------------|---------|
| Sem lista, sem produto | Seed Launch | Desafio gratuito | Baixo risco, valida antes de investir |
| Lista pequena (500-2K), sem ads | PLF Gratuito | Webinar ao vivo | Usa a lista que tem, custo zero |
| Lista grande (5K+), com budget | PLF Pago | Webinar automatizado | Estrutura comprovada, escala |
| Produto digital R$27-197 | Low Ticket (Tripwire) | Perpétuo | Adquire compradores, paga tráfego |
| Curso/mentoria R$500-2K | PLF Pago ou Desafio | Webinar | Estrutura de lançamento ou engajamento |
| Mentoria/consultoria R$3K+ | High Ticket | Evento Presencial | Venda consultiva necessária |
| Validar ideia nova | Seed Launch | Desafio gratuito | Feedback real antes de investir |
| Velocidade máxima (1-3 dias) | Campanha Rápida | Meteórico | Sprint de execução |
| Comunidade engajada no WhatsApp | Meteórico | Desafio | Lista quente, conversão rápida |
| Construir autoridade + vender | Webinar | Desafio | Formato educacional de alta conversão |
| Ticket altíssimo (R$10K+) | Evento Presencial | High Ticket | Pitch ao vivo, experiência imersiva |
| Escalar sem esforço contínuo | Perpétuo | Webinar automatizado | Roda no automático 24/7 |

## Pré-Perguntas Obrigatórias

Antes de recomendar qualquer funil, responder:

```
1. O produto já foi validado? (já vendeu pelo menos 1 vez?)
2. Qual é o tamanho da audiência atual? (lista, WhatsApp, seguidores)
3. Qual é o orçamento disponível para tráfego?
4. Qual é o ticket do produto?
5. O expert tem disponibilidade para aparecer ao vivo?
6. Existe histórico de campanhas anteriores?
7. Qual é o prazo?
8. Qual é a meta de faturamento?
```

## Decisão por Maturidade do Expert

```
INICIANTE (sem lista, sem produto):
→ Seed Launch → Meteórico → PLF Gratuito

INTERMEDIÁRIO (lista 500-5K, produto validado):
→ PLF Pago → Desafio → Webinar

AVANÇADO (lista 5K+, histórico de lançamentos):
→ PLF Pago + Perpétuo → Webinar Automatizado → High Ticket

EXPERT CONSOLIDADO (lista 10K+, múltiplos produtos):
→ Evento Presencial → Sala Secreta → Value Ladder completa
```

## Erros Comuns

| Erro | Consequência | Correção |
|------|-------------|----------|
| Funil complexo pra expert iniciante | Alto custo, baixa execução, frustração | Comece simples (Seed/Meteórico) |
| Oferta fraca com tráfego alto | CPL alto, conversão baixa, prejuízo | Corrija a oferta ANTES de escalar |
| Sem unique mechanism | "Mais do mesmo", prospect não vê diferença | Defina o mecanismo único antes de qualquer copy |
| Pular o pré-lançamento | Audiência fria, baixo engajamento | Aqueça 7-10 dias antes de abrir carrinho |
| Urgência falsa | Perde credibilidade | Urgência real: turmas, datas, bônus por tempo |
| Não ter downsell | Perde quem não comprou a oferta principal | Sempre tenha uma oferta B |
| Preço sem ancoragem | Prospect acha caro sem referência | Stack de valor com ancoragem antes do preço |
