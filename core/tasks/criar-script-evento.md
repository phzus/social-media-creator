# Task: Criar Script do Evento (Run of Show)

```yaml
task:
  name: criar-script-evento
  agent: pitch-master
  elicit: true
  output: "script-evento.md"
```

## Briefing Obrigatório (perguntar ao usuário)

Antes de criar o script, coletar:

1. **Nome do evento** — Como se chama a imersão?
2. **Tema/nicho** — Sobre o que é o conteúdo? (ex: copywriting, funis, tráfego)
3. **Oferta do pitch** — O que vai vender? (produto, preço, bônus, garantia)
4. **Público** — Quem é a audiência? Nível de consciência?
5. **Horário** — Confirmar 9h–17h com almoço 12h–14h (ou ajustar)
6. **Quantos blocos de conteúdo** — O que será ensinado em cada bloco?

## Workflow

### Step 1: Montar Estrutura do Dia

Criar o run of show com os seguintes blocos:

```
09h00–11h30  CONTEÚDO DENSO (manhã) — 3 blocos
11h30–12h00  SEEDING DE VENDA
12h00–14h00  ALMOÇO (2 horas)
14h00–15h00  CONTEÚDO PREMIUM (tarde)
15h00–15h30  PITCH DE VENDAS
15h30–17h00  MELHOR CONTEÚDO + REPEAT
```

### Step 2: Detalhar Cada Bloco

Para cada bloco, definir:
- Horário exato (início e fim)
- Título do bloco
- O que o expert vai ensinar/fazer
- Entregável para o aluno (se aplicável)
- Transição para o próximo bloco

### Step 3: Detalhar o Seeding

O seeding (11h30–12h) deve conter:
- Resumo do que aprenderam na manhã
- Frase-chave: "Imagina se você tivesse tudo isso implementado..."
- Mencionar o que alunos avançados já têm
- NÃO abrir oferta — só plantar a semente

### Step 4: Detalhar a Transição pro Pitch

Os últimos 15 minutos do conteúdo da tarde (14h45–15h):
- Transição natural: "Antes de continuar, preciso falar uma coisa importante..."
- Conectar o conteúdo da tarde com a oferta
- Gerar curiosidade sem entregar a oferta ainda

### Step 5: Detalhar o Pós-Pitch

O bloco 15h30–17h deve conter:
- O MELHOR conteúdo do dia inteiro
- Intercalar com soft sell: "Quem já comprou tem acesso a X..."
- Repetir CTA de forma natural a cada 15–20 minutos
- Celebrar compras no chat

## Output Format

Entregar como `script-evento.md` com:
- Tabela resumo do dia
- Detalhamento bloco a bloco
- Notas para o expert (o que falar, como se comportar)
- Transições escritas
