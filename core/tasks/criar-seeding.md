# Task: Criar Estratégia de Seeding nos CPLs

```yaml
task:
  name: criar-seeding
  agent: pitch-master
  elicit: true
  output: "seeding-cpls.md"
```

## Briefing Obrigatório

1. **Produto que será vendido** — Nome, promessa, preço
2. **Quantos CPLs/aulas** — Antes da imersão (ex: 3 CPLs de lançamento)
3. **Tema de cada CPL** — O que é ensinado em cada um
4. **Depoimentos disponíveis** — Formatos (vídeo, print, estudo de caso)

## Workflow

### Step 1: Identificar Momentos de Seeding
Para cada CPL, mapear 3-5 momentos naturais onde mencionar o produto

### Step 2: Criar Frases de Seeding
- Frases sutis que conectam o conteúdo ao produto
- NÃO é venda direta — é plantação de semente
- Exemplos: "Isso que eu mostrei é só a ponta do iceberg..."

### Step 3: Mapear Quebra de Objeções
- Cada tese levantada = 1 depoimento para comprová-la
- Variar formato: vídeo, print, estudo de caso

### Step 4: Funil de Comprometimento
- Antes e após cada CPL: ação de comprometimento
- Enquetes, desafios, tarefas práticas

## Output Format

`seeding-cpls.md` com tabela: CPL → Momento → Frase de Seeding → Depoimento
